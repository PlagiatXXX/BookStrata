# Документация Фронтенда

## 1. Технологический стек
- **React 19**: Используются хуки `use`, `useActionState` и оптимизации рендеринга.
- **TanStack Query (React Query)**: Управление серверным состоянием, кэширование и оптимистичные обновления.
- **TailwindCSS 4**: Стилизация с использованием JIT и плагина для Vite.
- **@dnd-kit**: Основная библиотека для реализации функционала Drag-and-Drop.
- **Framer Motion**: Анимации интерфейса.
- **Three.js / @react-three/fiber**: 3D визуализация книг.

## 2. Основные компоненты

### TierList Editor (`src/pages/TierListEditorPage`)
Центральный узел приложения. Реализует сложную логику перетаскивания книг между уровнями (Tiers) и списком нераспределенных элементов (Unranked Items).

- **TierGrid**: Контейнер для всех строк тир-листа.
- **TierRow**: Отдельная строка уровня. Содержит заголовок уровня и зону сброса для книг.
- **UnrankedItems**: Пул книг, которые еще не были распределены по уровням.

### Community (`src/pages/CommunityPage`)
Раздел для просмотра публичных тир-листов и шаблонов.
- **TemplateGrid**: Отображение доступных шаблонов для быстрого старта.
- **BookScene**: 3D-сцена с книгой, использующая WebGL для интерактивного просмотра.

### Auth (`src/pages/AuthPage`)
Унифицированная форма входа и регистрации с переключением состояний без перезагрузки страницы.

## 3. Состояние и хуки

- `useAuthContext`: Глобальный контекст для управления сессией пользователя и правами доступа.
- `useTierList`: Основной хук для управления состоянием текущего тир-листа в редакторе.
- `useAutoSaveOptimized`: Интеллектуальное автосохранение, которое отправляет на сервер только разницу (diff) состояний.
- `useBookSearch`: Интеграция с API поиска книг для динамического наполнения редактора.

## 4. Стилизация и UI-Kit (`src/ui`)
Мы придерживаемся принципа атомарного дизайна. Все базовые элементы управления находятся в папке `src/ui`:
- `Button`: Кнопки с поддержкой различных вариантов (primary, ghost, danger) и состояний загрузки.
- `Input` / `Textarea`: Поля ввода с анимированными фокусами.
- `Modal`: Доступные модальные окна с использованием порталов.
- `TierLabel`: Редактируемые метки уровней с выбором цвета.

## 5. Лучшие практики
- **Производительность**: Используется `React.memo` для предотвращения лишних рендеров в тяжелых списках книг.
- **Доступность (A11y)**: Все интерактивные элементы имеют соответствующие ARIA-атрибуты и поддерживают навигацию с клавиатуры.
- **Типизация**: 100% покрытие кода типами TypeScript, что исключает ошибки во время выполнения.
