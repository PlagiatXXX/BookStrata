## 🔍 Критические "заболевания" и рецепты

### 1. 🧪 Тестовая импотенция (Vitest 4.0.18 Compatibility)
**Симптомы:** Тесты написаны (~145+), но не запускаются из-за изменений в формате импортов или конфигурации пула в Vitest 4.x.
**Рецепт:**
- **Вариант А:** Откат Vitest до версии 3.x для стабильности.
- **Вариант Б:** Обновление `vitest.config.ts` (настройка `poolOptions`, замена `forks` на `threads` для стабильности на слабых машинах).
- **Вариант В:** Миграция всех spec-файлов на новый формат API Vitest 4.

### 2. 🔐 Склероз авторизации (JWT Expiry)
**Симптомы:** Отсутствие Refresh Tokens. Если токен истекает во время редактирования, пользователь может потерять прогресс при попытке сохранения.
**Рецепт:**
- Внедрить механизм автоматического обновления токенов в фоне через Axios/Fetch интерцепторы.
- Перейти на схему Access (short-lived) + Refresh (long-lived, cookie-based) tokens.

### 3. 📱 Drag-and-Drop Паралич (Mobile UX)
**Симптомы:** Редактор на смартфонах практически неиграбелен из-за конфликта жестов прокрутки и перетаскивания.
**Рецепт:**
- Добавить режим "Кнопочного управления" для мобильных разрешений.
- Использовать `activationConstraint` в `@dnd-kit` для более четкого определения начала перетаскивания (например, по длительному нажатию).
