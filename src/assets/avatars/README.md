# 📸 Локальные аватарки для TierMaker Pro

Этот каталог содержит пресеты аватарок, которые пользователи могут выбирать в профиле.

## 📁 Структура

```
src/assets/avatars/
├── cartoon/         # Мультяшные аватарки
├── fantasy/         # Фэнтези персонажи
├── professional/    # Профессиональные/деловые
├── minimalist/      # Минималистичные
├── animals/         # Животные
└── abstract/        # Абстрактные узоры
```

## 📋 Требования к файлам

### Формат
- **SVG** (рекомендуется) — масштабируются, маленький размер
- **PNG** (допускается) — минимум 256x256px

### Именование
```
avatar-1.svg
avatar-2.svg
avatar-3.svg
avatar-4.svg
avatar-5.svg
avatar-6.svg
```

### Размер
- **Оптимально:** до 50 KB
- **Максимум:** 100 KB
- **Размеры:** 256x256px или вектор (SVG)

## ➕ Как добавить аватарки

### Шаг 1: Подготовьте файлы
1. Создайте или выберите 6 аватарок для категории
2. Конвертируйте в SVG или PNG
3. Назовите файлы: `avatar-1.svg`, `avatar-2.svg`, и т.д.

### Шаг 2: Поместите в папку
Скопируйте файлы в соответствующую папку:
```
src/assets/avatars/cartoon/avatar-1.svg
src/assets/avatars/cartoon/avatar-2.svg
...
```

### Шаг 3: Обновите index.ts
В файле категории (`cartoon/index.ts` и т.д.) убедитесь, что импорты соответствуют именам файлов:

```typescript
import avatar1 from './avatar-1.svg';
import avatar2 from './avatar-2.svg';
// ... и так далее до avatar6
```

### Шаг 4: Перезапустите сервер
```bash
npm run dev
```

## 🎨 Категории

| Категория | Описание | Примеры |
|-----------|----------|---------|
| **Мультяшные** | Яркие, дружелюбные персонажи | Мультяшные лица, эмодзи |
| **Фэнтези** | Магические существа, герои | Эльфы, волшебники, драконы |
| **Профессиональные** | Деловые аватарки | Люди в костюмах, офис |
| **Минимализм** | Простые геометрические формы | Круги, линии, паттерны |
| **Животные** | Различные животные | Коты, собаки, птицы |
| **Абстракции** | Узоры, градиенты, формы | Геометрия, космос |

## 💡 Советы

1. **Единый стиль** — аватарки в категории должны сочетаться
2. **Контраст** — хорошо видно на маленьком размере
3. **Универсальность** — подходят для разных пользователей
4. **Без текста** — текст плохо читается на аватарках

## 🔧 Технические детали

- Файлы импортируются через Vite
- SVG автоматически оптимизируются
- Кэширование в localStorage (30 дней)
- Fallback на инициалы при ошибке загрузки

## 📊 Текущее количество

- **6 категорий** × **6 аватарок** = **36 всего**
