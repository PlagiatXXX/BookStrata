# 📊 Глубокий аудит и стратегия развития BookStrata Pro

**Дата:** 01 April 2026
**Аудитор:** Senior Fullstack Developer
**Цель:** Трансформация BookStrata Pro в ведущую платформу для книжных блогеров и читателей СНГ.

---

## 1. 🏗️ Техническая оценка (Architecture Review)

### Текущее состояние
- **Стек:** React 19 + Tailwind 4 (FE), Fastify 5 + Prisma (BE). Это "state-of-the-art" стек, обеспечивающий отличную производительность и типизацию.
- **БД:** PostgreSQL с четкой схемой. Присутствуют индексы на критических путях (`BookPlacement`, `Tier`).
- **Тесты:** 292 теста — отличный показатель стабильности.

### Критические рекомендации
1. **Синхронизация типов:** Вынести общие типы (DTS) в папку `shared/` для предотвращения рассинхронизации API и фронтенда.
2. **Refresh Tokens:** Сейчас используется один JWT. Для безопасности и долгоживущих сессий (важно для мобильных приложений) нужен механизм Access + Refresh Tokens (в HttpOnly cookies).
3. **Оптимизация Google Books:** Внедрить прокси-кеширование запросов к Google Books на стороне бэкенда для экономии лимитов и ускорения поиска.

## 🎨 2. UI/UX Аудит: Neo-Brutalism & Engagement

### Визуальный стиль
Проект использует **Neo-brutalism** (жирные границы, высокая контрастность, Space Grotesk). Это отличный выбор для выделения на фоне "мягких" конкурентов вроде Goodreads.

### Предложения по улучшению UX
1. **Интерактивный Dashboard:** Сейчас главная страница — это список. Нужно добавить "Виджеты достижений" и "Книгу месяца" в верхнюю часть.
2. **Полировка Editor:**
   - Добавить **"Quick Actions"** для книг (например, кнопка "Переместить в S" без перетаскивания).
   - **Undo/Redo:** Обязательный функционал для сложного редактора.
3. **Мобильный UX:**
   - Поскольку drag-and-drop сложен на мобилках, внедрить **"Tap-to-Place"** режим: нажимаем на книгу -> выбираем тир из списка.

## 🚀 3. Новые функции (Идеи для вовлечения)

### А. Социальные механики
1. **Публичная лента (Community Feed):**
   - Не просто шаблоны, а живые тир-листы пользователей.
   - Сортировка: "Тренды", "Новое", "Самое обсуждаемое".
2. **Система "Форков" (Forks):**
   - Кнопка "Создать свой на основе этого".
   - Копируется список книг, но обнуляются их позиции. Это создает виральность.
3. **Лайки и Комментарии:**
   - Возможность обсудить, почему "Преступление и наказание" попало в тир C.

### Б. Геймификация (Game-Loop)
1. **Достижения (Achievements):**
   - *«Библиофил-новичок»*: Создал первый тир-лист.
   - *«Классический гурман»*: Заполнил тир-лист с 10+ книгами классики.
   - *«Трендсеттер»*: Твой шаблон использовали 100 раз.
2. **Уровни профиля:**
   - Опыт (XP) за каждое действие (добавление книги, лайк, создание шаблона).
   - Визуальные лычки (badges) рядом с ником.
3. **Книжные челленджи:**
   - "Прочитать и ранжировать 12 новинок года".
   - Общий прогресс-бар челленджа в профиле.

## 📸 4. Инструменты для Блогеров (Export & Share)

### Эстетика экспорта
1. **Кастомные темы для PNG:**
   - *Minimalist*: Чистый белый/черный фон, тонкие линии.
   - *Vintage Paper*: Текстура старой бумаги, сепия.
   - *Cyberpunk*: Неоновые границы, темный фон.
2. **Водяные знаки:**
   - Настраиваемый текст (ник блогера) в углу изображения при экспорте (Pro-фича).
3. **Embed Виджеты:**
   - Генерация iFrame кода для вставки интерактивного тир-листа в статью на Teletype, VC.ru или личный блог.
